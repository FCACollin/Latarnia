The ROC is a general representation of a binary classifier: it accounts for the sensitivity (the proportion of real positive case detected), the false alarm rate (1 - sensitivity), the general performance via the area under the ROC curve (AUC).

The minimal ROC representation is simply a line plot representing the sensitivity as a function of the false alarm rate.

Aesthetic can help in the reading: the area under the curve is of interest, therefore it can be colored, the bisector delimite the reference level to compare the model to (flipping coin model).

The use of the model for diagnostic demands to define a threshold, various rational can be used, in the example two thresholds were defined: the minimal error, the 90% detection of positive case.

The plot can be further personalised manipulating the theme locally to address non-general specifications.

Finally, a last piece of annotation: the AUC given as an interval.
